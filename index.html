About

This website provides an online interface to OpenJML and was developed to simplify its use in a course at Chalmers university.

The web-interface can only handle one file at a time (packages are not supported) and has a timout of 10 seconds for syntax checking and 30 seconds for ESC with Yices 2.

If you find that the website malfunctions, please contact me about it and specify which browser you are using:
vandeba (a) chalmers.se

This service is provided as is. Information you enter while using this site is not stored.
